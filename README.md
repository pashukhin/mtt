# mtt

Минималистичный файловый таск-трекер для кодовых агентов и людей.

- Go CLI: `mtt add`, `mtt list`, …
- Хранение — YAML, один файл на задачу, каталог `.mtt/` в проекте (git-friendly, без демона).
- Иерархия эпик → задача → подзадача (`e1_t3_s2`), зависимости, типы задач с flow, комментарии.

Статус: **фаза 0 (каркас)**. План и архитектура — в [DESIGN.md](DESIGN.md),
правила работы — в [AGENTS.md](AGENTS.md).

## Сборка

```bash
make build      # -> ./bin/mtt
make check      # гейт: fmt + vet + lint + test -race + build
```
