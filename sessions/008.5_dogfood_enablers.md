# 008.5 — dogfood enablers (chore)

Status: done   ·   Branch: `feat/s008.5-dogfood-enablers`

## Target

Three small, unambiguous enablers batched into one session, unblocking self-hosting (dogfood, s009):
`mtt rm <id>` (hard-delete), `--depends-on` on `mtt add`, and packaging (version stamping + `make smoke`).
Plus a uniform exit-4 (not-found) taxonomy so an agent can branch on the code.

## Scope

- **In:**
  - `pkg/mtt`: base-port `TaskStore.Delete(id) error` (the *D* in CRUD; `ErrNotFound` when absent).
  - `adapter/yaml`: `Store.Delete` = `os.Remove` (atomic unlink; `ErrNotFound` when absent).
  - `core`: `Remover` usecase (reject-if-referenced via `Index`+`DepGraph`, deduped; `--force` overrides;
    no clock); `AddParams.DependsOn` validated + deduped in `Adder.resolveDeps` before `Create`; wrap
    `ErrNotFound` in `transition`/`dependency`/`add` not-found paths.
  - `cli`: `mtt rm <id> [--force]` (explicit id, clears stale `current`); `exitCode` `ErrNotFound → 4`;
    `taskNotFound` helper + wrap in `show`/`edit`/`tree`/`use`/`dep` (uniform exit-4); `add --depends-on`
    (StringSlice); `version.go` → stdout.
  - packaging: Makefile `VERSION`/conditional `LDFLAGS` + `make smoke`; bump `0.8.0-dev` → `0.8.5-dev`.
- **Out (deferred):** bulk `rm` / task-set selector / stdin `-` / `--ids` / `--dry-run` → **s008.9**; cascade
  delete, interactive confirmation, `--json` echo on `rm`, current resolution on `rm` (all rejected in
  brainstorm); mandating `--who`/`--why` on flow-bypassing ops (separate think-item); goreleaser (later).

## Open questions (resolved in brainstorm — see the spec)

Spec: [../docs/superpowers/specs/2026-07-07-session-008.5-dogfood-enablers-design.md](../docs/superpowers/specs/2026-07-07-session-008.5-dogfood-enablers-design.md).

1. **Delete port** — base `TaskStore.Delete` (a store op, not an embedded field; every adapter implements it).
2. **Dangling refs** — reject-if-referenced; `--force` overrides (leaves dangling, which the system tolerates).
3. **Safety** — delete immediately (agent-facing: no interactive prompt); no attribution (no history to sign).
4. **Not-found exit code** — `4`, applied **uniformly** to all single-task-by-id commands (subagent review).

## Acceptance (must pass)

- **rm:** `add A,B` → `dep add t2 t1` → `rm t1` rejected (names t2) → `rm t1 --force` deletes → `rm t9` exit 4.
- **--depends-on:** `add B --depends-on t1` sets `depends_on: [t1]`; `add C --depends-on t99` errors, none created.
- **packaging:** `make smoke` installs + runs `version`/`--help`; `make smoke VERSION=…` stamps via ldflags.
- **e2e:** `rm.txt`, `add_depends_on.txt`; `make check` green.
